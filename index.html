Text Entry Assistive Cloud-Based Platform (Alpha 1.0)

The primary goal of this alpha prototype prototype is to create a real-time text analysis engine that based on inspection of entered text offers predictive suggestions to the end user.
The project claims that the smart predictive technologies can speed up the text entry process, as well as improve user experience and overall user productivity.
